# Release Notes

## v1.0.0 (задание 1)

- Реализован краулер: скачивание до 100 страниц по списку URL
- Сохранение HTML без очистки разметки (кодировка UTF-8)
- Генерация index.txt (номер<TAB>url)
- Валидация результата (команда validate)
- Упаковка архива для сдачи (команда package → submission.zip)
- Пример мини-выкачки в examples/

## v2.0.0 (задание 2)
- Реализована токенизация сохранённых документов
- Выделение слов из каждого выкачанного файла
- Удаление дубликатов, служебных слов, чисел и текстового мусора
- Фильтрация некорректных токенов (HTML-фрагменты, смешанные символы)
- Формирование списка уникальных токенов для каждого документа (<id>_tokens.txt)
- Лемматизация токенов
- Группировка токенов по леммам для каждого документа (<id>_lemmas.txt)
- Сохранение результатов обработки в отдельные файлы для каждой выкачанной страницы

## v3.0.0 (задание 3)
- Построен инвертированный индекс терминов
- Сохранение индекса в отдельный файл
- Реализован булев поиск по индексу
- Поддержка операторов AND, OR, NOT
- Поддержка сложных логических запросов со скобками
- Обработка пользовательского поискового запроса
- Поиск документов по введённой строке запроса
